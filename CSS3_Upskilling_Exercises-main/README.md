# CSS3 Upskilling Exercises

This repository contains solutions for the CSS3 exercises completed as part of the **Cognizant Digital Nurture 5.0 Upskilling Program**.

## Topics Covered

* CSS Introduction
* CSS Syntax and Comments
* Inline, Internal, and External CSS
* CSS Selectors
* Colors and Backgrounds
* Fonts and Text Styling
* Hyperlinks
* Lists and Tables
* CSS Box Model
* Margin, Padding, Border, and Outline
* Visibility and Display Properties
* Multi-Column Layouts
* Media Queries
* Responsive Web Design (RWD)

## Technologies Used

* HTML5
* CSS3
* Visual Studio Code
* Google Chrome DevTools

## Learning Outcomes

* Applied different CSS styling techniques
* Worked with selectors and properties
* Designed responsive web pages
* Implemented layouts using the CSS Box Model
* Created device-friendly interfaces using Media Queries
