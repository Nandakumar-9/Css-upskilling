Exercise 11: Debug and Test with DevTools and VS Code

Objective:
To test and debug the website using Microsoft Edge DevTools.

Tasks Performed:
1. Opened the website in Microsoft Edge.
2. Opened Developer Tools using Inspect.
3. Tested the website in mobile view using Device Toolbar.
4. Inspected HTML and CSS using the Elements tab.
5. Verified applied CSS styles in the Styles section.
6. Opened the Network tab and checked file loading.
7. Confirmed styles.css and image files loaded successfully.

Result:
The website was tested successfully on different screen sizes and all resources loaded correctly.
